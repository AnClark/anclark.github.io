Markdown 拓展

Hexo默认的Markdown渲染器hexo-renderer-marked，相比于Typora等Markdown环境，还是有很多欠缺的地方。幸好，有些专门的插件能对症下药解决问题。

hexo-reference

添加引文支持。

引文格式为：假设a为整数，正文适当部分插入[^a]，然后在文章结尾插入[^a]: 引文内容。

项目地址：https://github.com/kchen0x/hexo-reference

hexo-toc

添加目录支持。

只要插入占位符[TOC]，就可以生成目录。

项目地址：https://github.com/bubkoo/hexo-toc

hexo-math

添加 MathJax 支持，可在文章中插入 LaTeX 与 KaTeX公式。
